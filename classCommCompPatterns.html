◆ ProcCommDict()

void CommCompPatterns::ProcCommDict ( const CommDict::const_iterator &  iter)
private

Process one element/iterator of CommDict.

The key of a CommDict determines the source and target partitions, the Backpatch determines what needs to be done. The SrcType::src_mbuf_idx in the index added to Buffers::sbuf, the SrcType::type determines which what needs to be updated (Mcsr::mcol for kMcol, MpiBuffers::init_idcs for kInitIdcs), and the set mapped to by the backpatch contains all the indices which need to be updated.

Parameters
iterIterator representing one element in CommDict.
